# T008: Create Workflow Diagram

## Objective

Add an ASCII diagram illustrating the Plan→Act→Observe→Fix workflow.

## Setup

AGENT_OPERATING_MODEL.md with text description of workflow.

## Task Description

1. Read the workflow description
2. Design a clear ASCII diagram
3. Add diagram to appropriate section
4. Ensure alignment and readability
5. Add caption/legend if needed

## Acceptance Criteria

- [ ] Diagram accurately represents workflow
- [ ] Shows all four phases
- [ ] Includes escalation path
- [ ] Includes checkpoint integration
- [ ] Renders correctly in markdown
- [ ] Is readable at standard terminal width

## Constraints

- ASCII only (no unicode box drawing)
- Max width 70 characters
- Must fit existing document flow

## Expected Duration

20-30 minutes

## Difficulty

Medium
