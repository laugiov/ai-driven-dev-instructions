# T001: Fix Broken Link in README

## Objective

Identify and fix a broken internal link in the README.md file.

## Setup

A test repository with a README.md containing a broken link to a non-existent file.

## Task Description

1. Read README.md
2. Identify the broken link
3. Determine the correct target
4. Fix the link
5. Verify the fix

## Acceptance Criteria

- [ ] Broken link identified correctly
- [ ] Link fixed to point to valid target
- [ ] No other content modified
- [ ] Commit message is descriptive

## Constraints

- Only modify README.md
- Do not create new files
- Single commit

## Expected Duration

5-10 minutes

## Difficulty

Easy
