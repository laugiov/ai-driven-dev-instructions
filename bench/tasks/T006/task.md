# T006: Create Issue Template

## Objective

Create a GitHub issue template for documentation improvement requests.

## Setup

Existing issue templates in .github/ISSUE_TEMPLATE/

## Task Description

1. Review existing issue templates
2. Create new template: docs-improvement.yml
3. Include fields for: section, current state, proposed improvement
4. Add appropriate labels
5. Follow YAML schema for GitHub issue templates

## Acceptance Criteria

- [ ] Template uses correct YAML format
- [ ] Includes all required fields
- [ ] Labels are appropriate
- [ ] Pre-submission checklist included
- [ ] Consistent with existing templates

## Constraints

- Must be valid GitHub issue template YAML
- Template under 60 lines
- No modifications to existing templates

## Expected Duration

20-25 minutes

## Difficulty

Medium
