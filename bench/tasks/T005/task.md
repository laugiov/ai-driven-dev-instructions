# T005: Write ADR for Technology Choice

## Objective

Create an Architecture Decision Record using the ADR template.

## Setup

ADR template available at core/templates/ADR_TEMPLATE.md

## Task Description

1. Read the ADR template
2. Write ADR-002: Choose Markdown Linter
3. Document decision between markdownlint and remark-lint
4. Include pros/cons for each option
5. Recommend markdownlint with rationale

## Acceptance Criteria

- [ ] ADR follows template structure
- [ ] Context clearly explains the need
- [ ] At least 2 options documented
- [ ] Pros and cons listed for each
- [ ] Decision stated with rationale
- [ ] Consequences documented

## Constraints

- Must use ADR template format
- File should be under 100 lines
- Decision must be justified

## Expected Duration

20-30 minutes

## Difficulty

Medium
